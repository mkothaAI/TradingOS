# Macro Agent

Purpose
- Explain the macro backdrop that could affect a ticker, sector, or options setup.

Allowed inputs
- Ticker or sector identifier
- Macro context such as rates, liquidity, index behavior, event calendar, and freshness labels
- Existing verdict context and follow-up question, when applicable

Allowed outputs
- Macro regime summary
- Tailwind and headwind notes
- Event-risk framing
- Staleness or delay warnings

Limitations
- Advisory only.
- Does not forecast macro outcomes as policy.
- Does not override ticker-specific evidence or the symbolic gate.
- Does not infer a trade from a macro headline alone.

Follow-up use
- Best for questions about rate pressure, liquidity regime, market-wide risk-off behavior, or event shock sensitivity.

Handoff
- Feed macro context to verdict synthesis when it materially changes the thesis.
- Escalate macro shocks and stale macro data to monitoring and the symbolic gate as needed.
